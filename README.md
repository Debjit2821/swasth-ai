# 🩺 SWASTH AI - Medical Monitoring & Appointment Scheduling System

SWASTH AI is an AI-powered healthcare platform designed to assist users with medical support, appointment scheduling, health monitoring, and secure medical report management.  
The system combines Artificial Intelligence and secure healthcare technologies to create a smart, reliable, and user-friendly healthcare ecosystem.

---

## 🚀 Features

- 🤖 AI-powered medical chatbot assistance
- 📅 Smart appointment scheduling system
- 🏥 Real-time medical monitoring
- 🔒 Secure medical report management
- 🛡️ Tamper-proof medical report verification
- 👤 Secure user authentication system
- 💬 Medical-related queries only restriction
- ⚡ Fast AI responses powered by Groq AI
- 🎨 Modern and responsive user interface
- 📲 Automated healthcare notifications

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/SWASTH-AI.git
cd SWASTH-AI
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / Mac

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python app.py
```
