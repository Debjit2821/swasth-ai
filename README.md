# 🩺 SWASTH AI - Medical Chatbot

SWASTH AI is an AI-powered medical chatbot built to assist users with healthcare-related queries.  
The chatbot is designed to provide responses only for medical-related questions and avoid unrelated conversations.

## 🚀 Features

- 🤖 AI-powered medical assistant
- 💬 Real-time chatbot interaction
- 🏥 Medical queries only restriction
- 🔒 Safe and focused response system
- ⚡ Fast responses using Groq AI
- 🎨 Simple and clean UI
- 📱 Responsive design
